# erosbag-core

Core primitives and operations for processing ROS2 bags.

> [!WARNING]  
> The library is at an early stage of development.

## Setting up Diesel

```bash
cargo install diesel_cli

cd erosbag-core
echo DATABASE_URL=sqlite://localhost/diesel_demo > .env
```
